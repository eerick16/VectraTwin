# Simulation Engine

## ETA Formula

ETA = Distance / Speed

Adjusted ETA = ETA + Delays

---

Fuel Cost +
Labor Cost +
Handling Cost +
Insurance +
Penalties

---

## Carbon Formula

CO2 =
Distance × Emission Factor

---

## Risk Formula

Risk Score =
Weather +
Congestion +
Asset Condition +
Route Criticality
